== Description ==

The New way of sharing. Let your visitors take your site with them!. Contains an Advanced Recent Post Widget. Controllable excerpt length, Custom Title, Templates; Usable on any website or blog.

If you have a website and a wordpress blog you might want to show off your latest blog posts on your website..

You can add the blog2widget widget into your sidebar and it will display your latest blog posts and also provides a code snippet which they can paste into their site and they will be able to show your latest blog post on their website.

Just paste in the javascript code into your website and take advantage of the share options as well as show off your latest posts.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

Make sure you save the settings before you start using the plugin
